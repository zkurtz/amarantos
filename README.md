# Amarantos

Tips and tools for personal wellness and longevity.

## About the Name

*Amarantos* (ἀμάραντος) is Greek for "unfading" or "immortal." The word gives us *amaranth*—a flower known for retaining its color even when dried. In ancient Greece, amaranth symbolized immortality. While true immortality remains elusive, this repo collects practical resources for living a longer, healthier life.
